# Vuser Value Proposition

| AI Agents | Web Publishers | Users |
| :--- | :--- | :--- |
| 1. **List all tabs** with optional MCP if available<br>2. **Call MCP tools** in a site | 1. **Control Access**: Choose to allow or deny AI calls<br>2. **Monetize**: Serve AI requests of virtual-user for a fee (0 or more) | 1. **Save Time**: Avoid unnecessary browsing and get more value from the internet<br>2. **Earn Rewards**: Adds VOC to wallet if interested in high valued browsing |

## Blockchain Ecosystem

*   **Blockchain** provides the infrastructure.
*   **Publishers** $\rightarrow$ Ask coins from AI agents for accessing MCP.
*   **Users** $\rightarrow$ Pay coins if interested in high value agentic browsing.
